# Jira Project Management

## Overview

The project was managed using Agile methodology (Scrum) to ensure efficient collaboration and task tracking.

---

## Tool Used

Jira was used to:

- organize tasks
- track progress
- assign responsibilities
- manage sprints

## Sprint Organization

### Sprint 1
- Data collection
- GitHub setup
- Identification of data sources

### Sprint 2
- Data cleaning
- Data preparation
- Database setup

### Sprint 3
- Data analysis
- Dashboard creation
- Visualization and results

---

## Team Roles

- Product Owner → defines requirements
- Data Developer → collects and processes data
- Analyst → performs analysis
- Documentation → prepares reports and presentation

---

## Benefits of Using Jira

- Better organization of tasks
- Clear distribution of work
- Efficient tracking of project progress
- Improved team collaboration
