# Final Report

## 1. Introduction
This project focuses on building a student decision support platform based on data analysis.

## 2. Problem Statement
Students often struggle to choose housing near universities while considering price, distance, and access to services such as libraries.

## 3. Objectives
- Compare student housing
- Analyze access to universities and libraries
- Provide recommendations based on data

## 4. Methodology
The project follows an Agile methodology with three sprints:
- Sprint 1: data collection
- Sprint 2: cleaning and integration
- Sprint 3: analysis and visualization

## 5. Data Sources
The project uses data from APIs, open data, and scraped sources.

## 6. Database Design
The project models universities, libraries, and apartments in a structured schema.

## 7. Data Analysis
The project uses descriptive statistics and indicators to compare different options.

## 8. Visualization
The results are presented through charts, dashboards, and maps.

## 9. Conclusion
This project demonstrates how data can help students make better decisions about housing and academic life.
