# Student Decision Platform - Project Overview

## Introduction

This project aims to help students choose the best housing based on proximity to universities, access to libraries, and cost of living.

---

## Problematic

Students often struggle to find:
- affordable housing
- nearby universities
- accessible libraries

---

## Objectives

- Compare student housing
- Analyze rental prices
- Identify nearby libraries
- Provide decision support

---

## Project Steps

### 1. Data Collection
- OpenStreetMap (API)
- Open data (Paris)
- CSV files

### 2. Data Cleaning
- Remove missing values
- Normalize data
- Prepare datasets

### 3. Database
- Tables: universites, bibliotheques, logements
- SQL schema created

### 4. Data Analysis
- Distance calculation
- Rent estimation
- Student score

### 5. Visualization
- Dashboard (Streamlit)
- Maps
- Charts

### 6. Decision Support
- Best housing recommendations
- Comparison between options

---

## Data Pipeline

Data → Cleaning → Database → Analysis → Visualization → Decision

---
