# Project Methodology

The project follows an Agile methodology using a simplified Scrum framework.

## Sprint 1
Data collection and repository setup

Tasks:
- Define project objectives
- Define user stories
- Create GitHub repository
- Collect initial datasets

## Sprint 2
Data cleaning and database integration

Tasks:
- Clean datasets
- Normalize data
- Design database schema
- Insert data into database

## Sprint 3
Analysis and visualization

Tasks:
- Perform data analysis
- Generate indicators
- Create visualizations
- Prepare presentation
