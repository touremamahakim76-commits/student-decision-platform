# Base de données

Ce dossier contient les fichiers SQL du projet.

## Contenu

- `schema.sql` : création des tables de la base de données
- `queries.sql` : requêtes SQL utilisées pour l’analyse

## Tables principales

- `universites`
- `bibliotheques`
- `logements`

## Objectif

Cette base de données permet de :

- stocker les données nettoyées
- structurer les informations du projet
- faire des analyses SQL
- alimenter le dashboard et les recommandations
